Technical Interview

Q1. Explain the concept of Object-Oriented Programming (OOP) in Java.

A1. OOP is a programming paradigm based on the concept of objects, which can contain data in the form of fields, and code in the form of methods. Key principles include encapsulation, inheritance, polymorphism, and abstraction.

Q2. What is the difference between an abstract class and an interface in Java?

A2. An abstract class can have both abstract methods (without implementation) and concrete methods (with implementation), while an interface can only have abstract methods (prior to Java 8). Java 8 onwards, interfaces can have default and static methods.

Q3. Write a Java program to reverse a string.

A3. 

public class ReverseString {
    public static void main(String[] args) {
        String original = "Hello";
        String reversed = new StringBuilder(original).reverse().toString();
        System.out.println("Reversed string: " + reversed);
    }
}

Q4. How does Java handle memory management?

A4. Java handles memory management through the use of an automatic garbage collector that deallocates memory for objects that are no longer referenced.

Q5. What are the main differences between JDK, JRE, and JVM?

A5. JDK (Java Development Kit) is used to develop Java applications and includes the JRE and development tools. JRE (Java Runtime Environment) is used to run Java applications and includes the JVM and standard libraries. JVM (Java Virtual Machine) is the engine that executes Java bytecode.

Q6. Explain the use of the 'final' keyword in Java.

A6. The 'final' keyword can be used to declare constants (variables that cannot be reassigned), prevent method overriding, and prevent inheritance by making a class final.

Q7. What is exception handling in Java?

A7. Exception handling in Java is a mechanism to handle runtime errors. It uses try, catch, finally blocks, and throws and throw keywords to manage exceptions.

Q8. Describe the Java Collections Framework.

A8. The Java Collections Framework provides a set of classes and interfaces for storing and manipulating groups of data as a single unit. It includes interfaces like List, Set, and Map, and classes like ArrayList, HashSet, and HashMap.

Q9. What is multithreading in Java?

A9. Multithreading is a Java feature that allows the execution of two or more threads concurrently. Threads are lightweight processes that share the same memory space, enabling parallel execution of code.

Q10. What is the difference between '== operator' and '.equals()' method in Java?

A10. The '==' operator compares references (memory addresses) to check if two objects are the same instance. The '.equals()' method is used to compare the actual content of objects for logical equality.
